# WEEK-09 Guide - Integrasi Dasar Gemini API

## Scope Minggu 9
- App bisa kirim prompt ke Gemini.
- App bisa menerima dan menampilkan respons.
- Menangani timeout, quota, dan network error.

## Langkah Pengerjaan Lengkap
1. Buat Gemini client dasar (HTTP request/response).
2. Implement method kirim prompt sederhana.
3. Tampilkan respons Gemini ke UI.
4. Tambahkan timeout pada request.
5. Tangani error quota dan network dengan pesan aman.
6. Uji minimal 3 skenario: sukses, timeout, gagal jaringan.

## Screenshot Wajib
- Screenshot request prompt berhasil.
- Screenshot respons berhasil di UI.
- Screenshot kasus timeout/error.
- Screenshot mapping pesan error ke user.

## Penjelasan Wajib per Screenshot
- Alur data request/response.
- Strategi error handling.
- Dampak ke pengalaman pengguna.

## Output Minggu 9
- LP7: contoh request/response, error handling, screenshot chat.
