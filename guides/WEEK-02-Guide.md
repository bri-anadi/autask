# WEEK-02 Guide - Setup Project Flutter

## Scope Minggu 2
- Setup project Flutter siap run.
- Menentukan dependency inti.
- Menetapkan struktur folder clean architecture.
- Menetapkan aturan branch/commit.

## Langkah Pengerjaan Lengkap
1. Inisialisasi project Flutter dan jalankan di emulator/device.
2. Tambahkan dependency inti: flutter_bloc, get_it, equatable, sqflite, path, intl.
3. Bentuk struktur folder `presentation`, `domain`, `data`.
4. Buat composition root dependency injection.
5. Buat dokumen aturan kerja Git (branching + commit convention).
6. Validasi project tetap berjalan setelah setup.

## Screenshot Wajib
- Screenshot app pertama kali running.
- Screenshot pubspec dependency terpasang.
- Screenshot struktur folder final.
- Screenshot contoh branch/commit sesuai aturan.

## Penjelasan Wajib per Screenshot
- Alasan memilih dependency.
- Fungsi setiap dependency utama.
- Alasan struktur folder dipilih.
- Dampak setup ke maintainability.
