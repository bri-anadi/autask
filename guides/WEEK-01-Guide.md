# WEEK-01 Guide - Perkenalan Proyek

## Scope Minggu 1
- Memahami problem statement Autask.
- Menentukan persona utama.
- Menyusun user story awal.
- Memisahkan fitur MVP vs non-MVP.
- Menetapkan risiko awal proyek.

## Langkah Pengerjaan Lengkap
1. Definisikan masalah utama yang ingin diselesaikan Autask.
2. Buat minimal 2 persona utama (contoh: mahasiswa aktif, freelancer).
3. Tulis 5-7 user story dengan format: "Sebagai ..., saya ingin ..., agar ...".
4. Kelompokkan semua fitur ke 2 kategori: MVP dan non-MVP.
5. Susun prioritas sprint awal berdasarkan urgensi dan effort.
6. Catat risiko teknis dan non-teknis awal.

## Screenshot Wajib
- Screenshot dokumen problem statement.
- Screenshot tabel persona.
- Screenshot user story list.
- Screenshot tabel MVP vs non-MVP.
- Screenshot prioritas sprint awal.

## Penjelasan Wajib per Screenshot
- Tujuan langkah.
- Aktivitas yang dilakukan.
- Keputusan yang diambil.
- Hasil akhir langkah.

## Output Minggu 1
- LP0 berisi: latar belakang, target user, fitur MVP, risiko.

## Format Pengumpulan
- Nama file: Kelompok_[Nomor_Kelompok].pdf
