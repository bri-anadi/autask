# WEEK-03 Guide - UI Dasar Task dan Notes

## Scope Minggu 3
- Membangun UI CRUD dasar task.
- Menyelesaikan flow list-detail-form.
- Menyiapkan interaksi tambah dan hapus task.

## Langkah Pengerjaan Lengkap
1. Buat halaman list task.
2. Buat form tambah task (judul + deskripsi).
3. Tambahkan aksi simpan task baru.
4. Tambahkan aksi delete task sederhana.
5. Buat halaman detail task dari item list.
6. Uji flow list -> detail -> kembali ke list.

## Screenshot Wajib

- Screenshot hasil task baru muncul di list.
- Screenshot aksi delete task.
- Screenshot halaman detail task.

## Penjelasan Wajib per Screenshot

- Alur interaksi user.
